## test task: GitHub GraphQL API

### Run on a local host

1. Clone the repository

```sh
git clone https://github.com/cantona94/github-graphql-api.git
```

2. Install NPM packages

```sh
cd github-graphql-api
```

```sh
npm install
```

3. Launch of the project

```sh
npm run dev
```
